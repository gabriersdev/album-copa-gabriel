# Instruções para gerar a base dos SCRIPTS

- Seguindo o arquivo `script.txt`, que tem o conteúdo de TODOS os exercícios, por favor separe cada exercício para produção de um arquivo posteriormente.
- O package em que os arquivos devem ser criados deve ser indicado no prompt.
- Os arquivos devem ser nomeados da seguinte forma: `Gabriel[Número do exercício em 2 caracteres. Se menor que 10, usar o zero à esquerda].java`.
- Cada exercício deve estar em um arquivo JAVA diferente, que contenha a estrutura base para executá-lo: 
    ```java
    // [NOTA como INSTRUÇÃO para produção do código] o nome do pacote deve ser o mesmo do package (diretório JAVA)  
    package exerciseXX;
    
    // [NOTA como INSTRUÇÃO para produção do código] o nome da classe deve ser o mesmo nome do arquivo
    public class GabrielXX {
        public static void main(String[] args) {
            // [NOTA como INSTRUÇÃO para produção do código] o enunciado vem aqui...
        }
    }
    ```
