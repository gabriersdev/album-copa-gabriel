# Instruções para produção de código com IA

- Declarar no início de cada script TODAS as variáveis que serão usadas em seus respectivos TIPOS.
- Ler o enunciado dos exercícios (no topo de cada UM) e fazer o que se pede de acordo com os enunciados. 
- Se as variáveis FOREM DO MESMO TIPO, listá-las na mesma linha.
- Se atentar ao enunciado e atender ao que foi proposto.
- Evitar comentários genéricos, característicos de vibe coding e comentar apenas o que for estritamente necessário.
- Seguir a "estética de código" padrão, conforme demonstrado em @/ai-instructions/example.java.
- Usar `easy returns` e técnicas de "estética `clean code`".
- Evitar usar tipos "sofisticados" de dados como StringBuilder. Dar preferencia por tipos primitivos e,
  quando houver necessidade de iteração, usar estruturas de repetição ALIADAS a estruturas de ARRAY básicas: vetores de tipos primitivos ou string.
- Os nomes das variáveis e funções deve ser em inglês.
- Usar métodos estáticos de suporte sempre que forem necessários para evitar repetição excessiva de código.
